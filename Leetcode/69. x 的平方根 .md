## 69. x 的平方根 

给你一个非负整数 `x` ，计算并返回 `x` 的 **算术平方根** 。

由于返回类型是整数，结果只保留 **整数部分** ，小数部分将被 **舍去 。**

**注意：**不允许使用任何内置指数函数和算符，例如 `pow(x, 0.5)` 或者 `x ** 0.5` 。

**示例 1：**

**输入：**x = 4
**输出：**2

**示例 2：**

**输入：**x = 8
**输出：**2
**解释：**8 的算术平方根是 2.82842..., 由于返回类型是整数，小数部分将被舍去。

**提示：**

- `0 <= x <= 231 - 1`

## Notes
使用循环不变量去**主动控制二分搜索**
1. f(left - 1) <= x
2. f(right + 1) > x
这是怎么推导出来呢？
其实就是因为, f(mid) > x之后, right = mid - 1 那么 mid = right + 1, 而之前 `f(mid)> x` 说明 `f(right + 1)> x`; 同理可得 `f(left - 1)<= x`. 这就是循环不变量.
同时最后我们看到退出条件是 `!(left <= right)` 也就是 `left > right`, 那么由于 mid 是两者的中间值, 肯定前一个状态是 `left == right`, 所以最后的状态就是 `left = right + 1`, 那么最后的 `f(left) = f(right + 1) > x`, 最后的 `f(right) = f(left - 1) <= x`, 说明最后 right 就是最后一个满足**小于等于** x 的坐标, 而 left 就是第一个大于 x 的坐标, 这就是**循环不变量**的魅力.

## Solution 
```cpp
#include <bits/stdc++.h>
using namespace std;

class Solution {
public:
	// 二分搜索
    int mySqrt(int x) {
        // f(left - 1) <= x
        // f(right + 1) > x
        int left = 0, right = x;
        while (left <= right) {
            long long mid = left + (right - left) / 2;
            if (mid * mid > x) {
                right = mid - 1;
            } else {
                left = mid + 1;
            }
        }
        return right; // 最后一个 i * i <= x的数
    }
};


```
