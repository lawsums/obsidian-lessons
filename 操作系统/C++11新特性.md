  - 原理: 
    Lambda 表达式是**编译器自动生成的匿名类（闭包类）**，运行时实例化为闭包对象

## 使用场景(举例)

|特性|核心使用场景|示例代码|
|---|---|---|
|初始化列表|统一初始化变量 / 容器|`vector<int> v{1,2,3}; int arr[]{4,5,6};`|
|auto|简化长类型（迭代器 / 模板）|`auto it = vec.begin();`|
|decltype|推导表达式类型、配合 auto 做泛型编程|`decltype(vec[0]) val = 10;`|
|range-for|遍历容器 / 数组|`for(auto& x : vec) x *= 2;`|
|nullptr|空指针赋值 / 函数传参|`char* p = nullptr; func(nullptr);`|
|Lambda|就地回调（算法 / 线程）|`sort(vec.begin(), vec.end(), [](int a, int b){return a < b;});`|
|智能指针|自动释放内存（避免内存泄漏）|`unique_ptr<int> p(new int(10));`|
|右值引用|移动语义（减少拷贝）、完美转发|`string s = move(string("hello"));`|
