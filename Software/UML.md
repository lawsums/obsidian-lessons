2. Visual Studio Code 通过 hpp2plantuml 导出 **puml** 文件然后通过 plantUML 可以将其转换为 UML 图

## PlantUML使用方法

``` bash
hpp2plantuml -i XXX.cpp/h -o XXX.puml
```
