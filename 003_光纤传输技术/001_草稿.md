1. PIN 工作原理
在 P 区和 N 区之间加入一个轻掺杂 N 型 I 层，I 层形成宽耗尽层；光生载流子在宽耗尽层之间加速运动，减少复合，从而<font color="#d83931">提高响应速度和转换效率</font>

- [x] APD 雪崩光电二极管的工作原理
   APD 雪崩二极管在 PN 结上加高额反向电压形成电场；光生载流子在强电场中加速，<font color="#d83931">碰撞晶格产生新电子-空穴对，新载流子再加速碰撞，循环形成雪崩式倍增</font>，使光电流在管内放大

3. 光纤非线性效应产生的原因和主要类型是什么？
   在强光强下，光纤的本征特征随光强变化发生了变化

4. 放大器的应用形式
	   1. 前置放大器：位于光接收机前端，放大微弱输入光信号
	   2. 功率放大器：提升输出功率
	   3. <font color="#d83931">在线放大器</font>：补偿信号的传输效率
	   4. <font color="#d83931">主放大器</font>：对前置放大后的信号进一步放大
5. EDFA 的结构
	1. 掺铒光纤：核心增益介质
	2. <font color="#d83931">泵浦光源</font>：提供能量
	3. <font color="#d83931">光耦合器</font>：将信号光与泵浦光耦合到掺铒光纤
	4. 光隔离器：抑制反射光，防止干扰泵浦光源或影响放大器稳定，保证光信号单向传输；
	5. <font color="#d83931">光滤波器</font>：滤除自发辐射噪声，提升输出信号信噪比
6. 光纤色散产生的原因是什么？
	   1. 模内色散
		   1. 材料色散
		   2. 波导色散
	   2. 模间色散
	      1. 模式色散（多模光纤）
	      2. 偏振模色散（单模光纤）
7. 单模光纤的特点
	   1. 仅传输基模 $LP_{01}$
	   2. <font color="#d83931">带宽大</font>，传输距离远，通信容量大
	   3. <font color="#d83931">纤芯半径小</font>

- [x] EDFA 的工作原理？
  当较弱的信号光和较强的泵浦光一起输入进EDF 时，泵浦光激活 EDF 中的铒粒子，在信号光子的感应下，铒粒子产生受激辐射，跃迁到基态，将一粒一粒的光子注入信号光中，完成放大作用。

- [x] FRA 的工作原理？
   FRA 利用光纤中的<font color="#d83931">受激拉曼散射效应（SRS）</font>，在非线性物质中，SRS 造成<font color="#d83931">入射光</font>的一部分的<font color="#d83931">功率</font>跃迁到另一个<font color="#d83931">光束</font>上

- [x] 拉曼光纤放大器的技术特点
	1. 带宽较宽
	2. 设计简单
	3. 低噪声

- [x] EDFA 的应用形式
	  1. 线路放大
	  2. 功率放大
	  3. 前置放大

- [ ] 光交叉连接器（OXC）的核心功能及组成部分是什么？
	在光层实现多个光纤链路之间波长信道的任意交叉连接。
	功能：
	1. 波长路由
	2. 波长交换
	3. 波长上下路
	组成：
	- 光交叉连接矩阵（核心）
	- 输入接口
	- 输出接口
	- 管理控制单元

- [ ] 波分复用（WDM）的工作原理及核心优势是什么？
	工作原理：<font color="#d83931">发送端</font>将不同波长的信号放到一个信道内传输，接收端解复用恢复原信号
	优势：<font color="#d83931">大幅提高光纤传输容量和效率</font> ~~最大程度利用一个波段~~

- [ ] 光隔离器的工作原理和作用？
	原理：基于<font color="#d83931">法拉第磁光效应</font>，实现单向传输
	作用：抑制反射光

- [ ] 光耦合器熔融拉锥法的制作过程？
	光纤去涂覆层靠拢，高温（~1000℃）熔融拉伸成双锥体，实时监控功率达耦合比后封装

- [ ] 时分复用（TDM）的工作原理及主要类型是什么？
	时间划分为TDM帧，多路信号按序占固定时隙周期性传输
	类型：
	1. 分组交错
	2. 比特交错
