## Chap3 光纤传输特性
1. 损耗系数的产生原因
	1. 吸收损耗：光能被介质吸收转热能，含材料固有本征吸收、已可忽略的杂质吸收、影响极小的原子缺陷吸收；
	2. 散射损耗：介质不均致光向非传输方向扩散，含固有瑞利散射、已可忽略的结构缺陷散射；
	3. 辐射损耗：弯曲破坏全反射，含宏弯（随曲率半径减小损耗剧增）、微弯（成缆随机小弯曲引起）损耗。这些机制共同形成损耗系数。
2. 光纤色散的分类
	   1.  色度色散（同一模式不同频率成分传速不同，分为材料色散/波导色散）
	   2. 模式色散（不同模式传速不同，分为模式/偏振模色散）。
3. 光纤色散的原因
	1. 光纤材料、波导结构和模式结构。（光纤固有的特性）
	2. 光信号具有一定的频谱宽度。（光源、调制过程）
4. 光纤色散产生的影响
   使光脉冲展宽，严重时脉冲重叠形成码间干扰，增加误码率，限制通信的传输容量和中继距离。

## Chap 5 光检测器与光接收机
1. 光电效应: 光照射半导体，光子被吸收使电子跃迁，产生电子 - 空穴对，在外电场下形成光电流，实现光转电。
2. PIN 光电二极管工作原理：P 型和 N 型材料间加轻掺杂 N 型 I 层，I 层形成宽耗尽层；光生载流子在耗尽层内建电场下快速分离移动，减少复合，提升响应速度和转换效率。
3. APD 雪崩光电二极管工作原理：P-N 结加高额反向电压形成强电场；光生载流子在强电场中加速，碰撞晶格产生新电子 - 空穴对，新载流子再加速碰撞，循环形成雪崩式倍增，光电流在管内放大，减少外部放大器噪声，提升接收机灵敏度。

## Chap 6 光放大器
1. 光放大器的工作原理
   通过电注入实现粒子数反转，入射光通过有源区时因受激辐射而放大。其发光的媒介是非平衡载流子（电子-空穴对）。其机理与半导体激光器相同。
2. 光放大器的分类：主流光放大器分三类掺铒光纤放大器（EDFA）、拉曼光纤放大器（FRA）、半导体光放大器（SOA）
3. 光放大器的应用形式
	   1. **功率放大器**：用于提升光信号的输出功率，增强信号的传输能力；
	2. **在线放大器**：部署在光纤传输链路中，补偿信号的传输损耗；
	3. **前置放大器**：位于光接收机前端，放大微弱的输入光信号，提升接收灵敏度；
	4. **主放大器**：对前置放大后的信号进一步放大，满足后续处理的功率需求。
4. EDFA (掺饵光纤放大器) 的基本结构
	1. **掺铒光纤（EDF）**：核心增益介质，光纤中掺杂的铒离子（Er³+）吸收泵浦光能量后被激发，通过受激辐射对信号光进行放大。
	2. **泵浦光源**：提供能量，通常采用 980nm 或 1480nm 波长的激光器，将铒离子从基态激发至高能级，为信号光放大提供条件。
	3. **光耦合器**：将输入的信号光与泵浦光耦合到掺铒光纤中，实现能量的注入与信号的传输。
	4. **光隔离器**：抑制反射光，防止反向传输的光干扰泵浦光源或影响放大器的稳定工作，保证光信号单向传输。
	5. **光滤波器**：滤除放大过程中产生的自发辐射噪声，提升输出信号的信噪比，保证放大后信号的质量。
5. RFA（拉曼光纤放大器）的工作原理：利用光纤中的受激拉曼散射效应，泵浦光注入传输光纤后，其能量通过拉曼散射转移给信号光，使信号光得到放大；传输光纤本身作为增益介质，信号光与泵浦光在光纤中共同传输，持续完成能量转移，实现分布式放大。
6. 拉曼光纤放大器技术特点：
	1. 增益波长由泵浦波长决定
	2. 分布式放大
	3. 有宽带放大潜力
7. 拉曼光纤放大器的应用场景：
	1. 长距离光纤通信系统
	2. 波分复用系统
	3. 光纤传感系统
	4. 无中继传输系统
