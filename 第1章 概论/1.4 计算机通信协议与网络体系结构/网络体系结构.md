## 教学模型 

1. 应用层 
2. 运输层
3. 网络层
4. 数据链路层
5. 物理层


| 协议    | 功能描述                                                                                                                             | 协议数据单元           |
| ----- | -------------------------------------------------------------------------------------------------------------------------------- | ---------------- |
| 应用层   | 解决通过应用进程间的交互来完成特定网络应用。<br>对于不同的网络提供不同的应用层协议，例如HTTP协议、SMTP协议等。                                                                    | 报文               |
| 运输层   | 负责进程之间基于网络的通信问题。<br>传输控制协议TCP--------------------------------------------<br>用户数据报协议UDP----------------------------------------- | <br>报文段<br>用户数据段 |
| 网络层   | 解决数据包在多个网络之间传输和路由问题。                                                                                                             | IP 数据报           |
| 数据链路层 | 解决数据包在一个网络或者一段链路上传输的问题。                                                                                                          | 帧                |
| 物理层   | 解决使用何种信号来表示比特 0 和比特 1 的问题。                                                                                                       | 比特               |
